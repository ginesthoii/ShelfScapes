# Book Nook Lighting Modules

Reusable Arduino `.ino` sketches for DIY book nook builds.
Each module can run standalone or be merged with the base controller.

| File | Description | Ideal For |
|------|--------------|------------|
| **BookNookLighting_Base.ino** | Universal on/off + sensor logic | All builds |
| **CandleFlicker.ino** | Simulated candle or torch flicker | Fantasy / Library |
| **BreathingLight.ino** | Pulsing glow effect | Magical or sci-fi builds |
| **MotionTrigger.ino** | Lights triggered by motion | Shelf or display setups |
| **DayNightCycle.ino** | Simulated sunrise/sunset cycle | Ambient environments |
| **RGB_Controller.ino** | Full-color Neopixel effects | Advanced scenes |

All scripts use safe 5 V logic, standard Arduino UNO/Nano pins, and minimal power.
